# CERF Donor Contributions Dashboard

# Project Description
This project is an interactive dashboard built using Streamlit to analyse CERF donor contributions.

# Dataset
The dataset is obtained from the Humanitarian Data Exchange and contains information about donor contributions over time.

# Tools Used
- Python
- Pandas
- Streamlit
- Matplotlib

# How to Run
1. Install required libraries:
   pip install streamlit pandas matplotlib
2. Run the app:
   streamlit run app.py
